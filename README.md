# 🎥 CCTV AI Surveillance System

## 📌 Project Description
This project is an AI-based CCTV surveillance system that uses computer vision to detect and monitor activities in real-time.

It is designed to simulate a smart security system using Python and deep learning techniques.

---

## 🚀 Features
- Real-time video surveillance
- Object / person detection using AI
- Integration with OpenCV
- Lightweight YOLO model (yolov8n)
- Simple and efficient backend

---

## 🛠️ Tech Stack
- Python
- OpenCV
- YOLO (Ultralytics)
- Django (Backend)
- SQLite Database
